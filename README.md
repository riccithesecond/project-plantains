# Project Plantains

Project Plantains is a modular, open-source initiative focused on building detection engineering workflows and tools. It includes detection content in Sigma, YARA, KQL, and SQL formats, as well as `Gh0st`, a lightweight custom SIEM for developing and testing detections.

## Components

- `detection-rules/`: Contains Sigma, YARA, KQL, and SQL rules.
- `gh0st/`: Custom SIEM platform with GUI and detection pipeline.
