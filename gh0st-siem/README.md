# Gh0st SIEM

**Gh0st** is a lightweight, modular SIEM platform built as part of Project Plantains. It allows users to ingest logs, run Sigma/KQL/YARA rules, enrich alerts with threat intel, and simulate SIEM workflows using a Streamlit GUI.

## Features

- Rule execution engine
- Threat intelligence enrichment
- Streamlit-based GUI
- Alert triage and visualization
